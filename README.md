# FastTello

DJITelloPy wrapped by FastAPI


## Usage

```sh
# clone this repository
$ git clone https://github.com/plop91/FastTello.git

# move to this repository
$ cd FastTello

# install dependencies
$ pip install -r requirements.txt

# move to directory
$ cd FastTello

# connect to Tello on WiFi

# run server
$ uvicorn FastTello:app --reload

# navigate to http://127.0.0.1:8000/docs in your browser to see the Swagger UI for the API
```
